4. WAP to accept 10 numbers from user and count even numbers.
Solution:
int count = 0;
		int[] m = new int[10];
		Scanner s = new Scanner(System.in);
		System.out.println("Enter 10 numbers: ");
		
		for (int i = 0; i < m.length; i++) {
			m[i] = s.nextInt();
		}
		
		for (int i = 0; i < m.length; i++) {
			if(m[i]%2 == 0)
				count++;
		}
		
		System.out.println(count + " even numbers found!");

Q. WAP to accept 7 boolean values from the user and count total no of false values.

Q. WAP to accept 10 symbols from the user and count only alphabets from it.
Solution:
int count = 0;
		char[] m = new char[10];
		Scanner s = new Scanner(System.in);
		System.out.println("Enter 10 symbols: ");
		
		for (int i = 0; i < m.length; i++)
			m[i] = s.next().charAt(0);
		
		for (int i = 0; i < m.length; i++)
				if( (m[i] >= 'A' && m[i] <= 'Z') || (m[i] >= 'a' && m[i] <= 'z'))
					count++;
		
		System.out.println(count);
