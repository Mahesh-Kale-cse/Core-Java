Stack class -> LIFO

Step 1: Create an object of Stack 
Step 2: add elements using push() method 
Step 3: remove elements using pop() method 
Step 4: Display all elements using SOP

Sample code:
       Stack<String> s = new Stack<String>();
       s.push("Mango");
       s.push("Apple");
       s.push("Cherry");
       s.push("Litchy");
       s.push("Stawberry");

       System.out.println(s);
       s.pop();
       System.out.println(s);
